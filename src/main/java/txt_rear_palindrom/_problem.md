
Input: `string`
Output: `boolean`

A palindrome is a string which is the same if read forwards or back.

Task: Write a function that determines if the characters in the string can be rearranged into a palindrome.

## Examples

+ "aa" -> true
+ "bca" -> false
+ "aaaabb" -> true
+ "aac" -> true
+ "caa" -> true
+ "madam" -> true
+ "madma" -> true
